# Order Kebonsari 001

Aplikasi manajemen menu dan order bahan makanan harian.

## Fitur
- Input menu per hari, kategori otomatis terdeteksi
- Rekapitulasi bahan yang dipakai di beberapa hari
- Pesan order siap copy-paste ke supplier
- Checklist order harian (tersinkron antar tab)
- Histori & duplikat minggu lalu
- Data tersimpan otomatis di browser (localStorage)

## Deploy ke Vercel

1. Upload folder ini ke GitHub (lihat panduan di bawah)
2. Buka [vercel.com](https://vercel.com) → Sign in with GitHub
3. Klik **Add New Project** → pilih repo ini
4. Klik **Deploy** — selesai!

URL akan jadi: `https://order-kebonsari-001.vercel.app`

## Cara upload ke GitHub

1. Buka [github.com](https://github.com) → **New repository**
2. Nama repo: `order-kebonsari-001`
3. Pilih **Public** → klik **Create repository**
4. Upload semua file ini (index.html, style.css, app.js)
5. Lanjut ke Vercel

## File
- `index.html` — struktur halaman
- `style.css`  — tampilan & tema
- `app.js`     — semua logika aplikasi
